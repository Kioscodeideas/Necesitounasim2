# SIMPASS — eSIM & SIM Afiliados

Landing page para programa de afiliados de eSIM y SIM de viaje.

## Estructura

```
simpass/
├── index.html       → Landing page principal
├── assets/css/      → Estilos adicionales (futuro)
├── assets/js/       → Scripts adicionales (futuro)
├── assets/img/      → Imágenes (futuro)
├── robots.txt       → SEO
├── sitemap.xml      → SEO (actualizar dominio)
└── README.md
```

## Cómo publicar en GitHub Pages

1. Crea cuenta en github.com (si no tienes)
2. New repository → nombre: simpass (o el que elijas)
3. Sube estos archivos (drag & drop en la web de GitHub)
4. Settings → Pages → Branch: main → Save
5. En 2 minutos tienes URL: https://tu-usuario.github.io/simpass
